# architecture_library/__meta

This folder contains **non-normative** CAF library metadata and library-improvement artifacts.

## Conventions

- `architecture_library/__meta/` is the **single** canonical meta root.
- The previous `architecture_library/_meta/` folder has been removed.

## Contents

- `contura_phase_index_v1.yaml` / `contura_artifact_index_v1.yaml`
  - Library indexes used by meta tooling.

- `caf_library__evals/`
  - Non-normative artifacts used to improve retrieval quality and authoring cues.
  - Example: `caf_pins_expected_patterns_v1.md` (cue-sweep guidance; **not** an inclusion list).
